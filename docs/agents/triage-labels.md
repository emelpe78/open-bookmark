# Triage Labels

Die Skills nutzen fünf kanonische **State**-Rollen. Diese Tabelle mappt sie auf die tatsächlich im Tracker verwendeten Zeichenketten (bei lokalem Markdown: Wert in der `Status:`-Zeile der Issue-Datei).

| Label in mattpocock/skills | Label in diesem Repo | Bedeutung                                    |
| -------------------------- | -------------------- | -------------------------------------------- |
| `needs-triage`             | `needs-triage`       | Maintainer soll bewerten                     |
| `needs-info`               | `needs-info`         | Warten auf Reporter/Infos                    |
| `ready-for-agent`          | `ready-for-agent`    | Für AFK-Agent spezifiziert                   |
| `ready-for-human`          | `ready-for-human`    | Menschliche Umsetzung nötig                  |
| `wontfix`                  | `wontfix`            | wird nicht bearbeitet                        |

**Kategorie-Rollen** (laut `/triage`-Skill): pro Issue eine Kategorie **`bug`** oder **`enhancement`** zusätzlich zur State-Rolle — im lokalen Setup z. B. als eigene Zeile oder im Frontmatter dokumentieren.

Rechte Spalte an echte Labels/Status-Werte anpassen.
